# Tree Segmentation App
A web-app based on Flask-React stack for segmenting trees from images

## Dependencies
1. Linux/Mac
2. Docker

## Running Server
**Linux/Mac**
```console
foo@bar:~$ sudo docker-compose up
```

## Stopping Server
**Linux/Mac**  
Use ```Ctrl+C```

## Notes
1. Always use ```Ctrl + C``` and not ```Ctrl + X```
